# HamsterWheel

A fundamental framework helps to create graphics application.

* Modules
  
  * Core
  
  * Graphics
    
    * D3D12
  
  * OS
    
    * Hardware Info
    
    * Thread
    
    * File System
    
    * Input Management
    
    * Windows Management

* Application Framework Layer
  
  * Links internal modules

* Applications
  
  * Links Application Framework
  
  * Links external modules such as plugins
